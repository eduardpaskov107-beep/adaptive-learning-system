Переменные и типы данных
Время изучения: 10-15 минут

Переменная - это именованная область памяти для хранения данных.

Пример создания переменных в Python:

name = "Анна"          # строка
age = 25               # целое число
price = 19.99          # число с плавающей точкой
is_student = True      # булево значение

Основные типы данных:

  • int - целые числа
  • float - числа с плавающей точкой
  • str - строки
  • bool - логические значения (True/False)
  • list - списки
  • dict - словари
